# Sistema de Orçamento de Aluguel - Imobiliária R.M

Este projeto é um sistema simples em Python para gerar orçamento de aluguel para diferentes tipos de imóveis.

## Funcionalidades

- Seleção de tipo de imóvel: apartamento, casa ou estúdio.
- Escolha da quantidade de quartos (1 ou 2).
- Adição de vaga de garagem ou estacionamento.
- Desconto especial para apartamentos sem crianças.
- Geração de orçamento com valor mensal e contrato imobiliário.
- Exportação do orçamento para arquivo CSV com 12 meses.

## Como executar

```bash
python EmpresaRM.py
```
